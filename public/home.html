Welcome to E-Gates!

What is e-Gates? e-Gates is a easy way to learn and understad how the logic gates work.
In addittion, e-Gates is an API! You will be able to make GET requests using your web browser or PostMan.

How can i use it? You can add the necessary paramethers to the main URL (https://e-gates.herokuapp.com), adding the gate name followeb by the inputs. Each gate can allow two inputs.

Allowed Logic Gates

  • AND
  • NAND
  • OR
  • NOR
  • XOR
  • NOT

Allowed Inputs

  • ON
  • TRUE
  • 1
  • FALSE
  • OFF
  • O (Zero)

For example, if you want to know how the XOR gate works you can add /XOR/ON/OFF:
https://e-gates.herokuapp.com/XOR/ON/OFF
